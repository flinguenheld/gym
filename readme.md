## gym

Maintain your memory level with gym !  
This small game allows you to practice mental arithmetic and more in the terminal.

### Installation

Copy **gym** in your **~/bin/** folder for instance and launch it !

`gym -h` for the command list.  
`gym --multiplication`  
`gym --addition`  
`gym --substraction`
